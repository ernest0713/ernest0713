六角學院 — 網頁切版直播班心得

Posted on 2020-12-28

當初為什麼報名這堂課？

雖然自己本身也有買六角的相關一系列課程，但是自己看影片上課做作業跟一群人一起做作業跟上課根本就不是一個回事，為了讓自己不在這麼懶惰，所以就決定參加網頁切版直播班了。

最喜歡直播班的哪些活動？

  1. 直播：最喜歡的是星期五的晚上8點的直播上課時光，看著一群人一起學習的氛圍真的很棒，在配上老師幽默的教學口吻，會讓人覺得上在久都不嫌多，甚至連上廁所都要小跑步(明明在家裡)。
  2. 助教分享：你以為上完課就了事了? 不！星期一還有助教分享經驗，讓你知道如何去完成這一個禮拜的作業！！
  3. 任務獎勵：我相信有獎勵可以增加人的動機，就像要多讚美、鼓勵小孩子一樣，完成一定程度的任務就可以領取相對應的報酬，就像玩遊戲一樣解任務，解越多領多，還有滿滿的成就感，好任務不解嗎?！更何況任務獎勵送的都可以抵掉學費了。

Git常用指令列表

Posted on 2020-04-14

紀錄一下常用的Git指令，方便查詢

git init

當創建一個新專案的時候，需執行git init 進行初始化的動作，才能進行後續操作，這會在專案內新增一個.git檔案

git status

查看目前檔案追蹤的狀況

git add .

將所有新增或修改過的檔案建立索引

git commit -m “說明”

提交更新紀錄，並設定說明更新內容

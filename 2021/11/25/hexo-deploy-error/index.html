hexo deploy error

Posted on 2021-11-25 Edited on 2021-12-01 Disqus:

hexo deploy -g時出現以下錯誤

hexo TypeError [ERR_INVALID_ARG_TYPE]: The “mode” argument must be integer. Received an instance of Object

出現以上的錯誤，後來查詢了一下是因為node.js版本過高導致
當時我的node.js版本是v14~

改由較為穩定的v12版本後就可以正常使用了

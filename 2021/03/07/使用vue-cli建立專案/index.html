使用vue-cli 3以上版本建立專案

首先執行　npm install -g @vue/cli 執行全域安裝

如果不確定之前有沒有在全域環境安裝過，可以執行npm ls -g @vue/cli確認
或者
執行vue –version 確認版本

建立專案

安裝完vue-cli 後，接著執行 “vue create 專案名稱”，重點是不能有大寫字母

以上述指令來說，hello-world就是我的專案名稱

如果有輸入大寫字母的話會報以下的錯誤
